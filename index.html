Vocabularies for Open Data for Universities

This page contains the list of vocabularies that are being created at Open University Data for the homogeneous publication of open data from universities in UniversiDATA.


Ontology Serialization License Language Links Description
Vocabulario para la representación de datos sobre los estudios y los estudiantes universitarios rdf+xml html CC-BY es Repository
Issues 		Vocabulario para la representación de datos sobre los estudios y los estudiantes universitarios. El vocabulario describe todo lo relativo a las titulaciones, las matrículas, el acceso y la movilidad de los estudiantes.
Vocabulario para la representación de datos sobre la economía y el patrimonio universitarios rdf+xml html CC-BY es Repository
Issues 		Vocabulario para la representación de datos sobre la economía y el patrimonio universitarios. El vocabulario describe todo lo que tiene que ver con los presupuestos de ingresos y de gastos universitarios.
Vocabulario para la representación de datos sobre recursos humanos universitarios rdf+xml html CC-BY es Repository
Issues 		Vocabulario para la representación de datos sobre recursos humanos universitarios. El vocabulario describe todo lo relacionado con los empleados universitarios.
Vocabulario para la representación de datos sobre la organización y los servicios universitarios rdf+xml html CC-BY es Repository
Issues 		Vocabulario para la representación de datos sobre la organización y los servicios universitarios. El vocabulario describe todo lo que hace alusión a los centros y departamentos universitarios.
